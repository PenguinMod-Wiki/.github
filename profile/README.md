<h2>PenguinMod Wiki</h2>

This is the GitHub profile of your friendly neighborhood PenguinMod Wiki. This is where MediaWiki plugins maintained by the PenguinMod Wiki for the PenguinMod Wiki are hosted.

All of these are open source, so feel free to use them on your own wikis, or to fork them to customize them for your own purposes; or if you want to help us out, please feel free to submit pull requests with fixes for bugs or annoyances.
